# Атомарные якоря требований L4

> Расширение `Conventions.md §7`. Сопоставляет числовые ID L4 из `Spec.md` с атомарными якорями вида `L4-<area>-<group>.<field>` и привязывает их к вкладке карточки и EditGroup. Используется для трассировки «ТЗ → поле → экран → группа правки».

Области (`<area>`):
- `pax` — пассажиропоток и размеры движения
- `cargo` — грузовые перевозки
- `infra` — инфраструктура (станции, депо, перегоны, capacity)
- `fin` — финансовая модель
- `socio` — социально-экономические эффекты
- `team` — участники и RACI
- `risk` — риски и меры реагирования
- `sys` — системные/общие требования

## 8. Правила использования якорей

- В новых местах документации и в коде ссылаться **на якорь**, а не на числовой L4.
- При расхождении формулировок числовой ID и якорь считаются одной и той же сущностью; якорь — каноничен.
- При появлении нового атрибута сначала добавляется якорь сюда, затем привязывается EditGroup, затем правится Spec.
